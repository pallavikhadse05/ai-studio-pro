AI Studio Pro is a free offline AI SaaS built with Streamlit, Ollama, and Python. It supports chatbot, AI image generation, and advanced text-to-speech with audio download and speed control. No API key required.
# 🚀 AI Studio Pro

An offline AI-powered SaaS application built using **Streamlit + Ollama + Python**.

It includes:
- 💬 AI Chatbot (Offline LLM via Ollama)
- 🎨 AI Image Generator (Free API)
- 🔊 Text-to-Speech AI with speed control
- ⬇ Audio download feature
- ⚡ Fully free (No API key required)

# ⚡ Features

## 💬 Chat AI
- Offline AI chatbot using Ollama
- Fast responses
- Memory-based session chat

## 🎨 Image Generator
- AI image generation using free API
- Image preview
- Download option

## 🔊 Audio AI
- Text to speech conversion
- Speed control slider
- Play audio in browser
- Download MP3 file

# 🛠 Tech Stack

- Python 🐍
- Streamlit 🎈
- Ollama (LLM) 🤖
- Pillow (PIL) 🖼
- pyttsx3 🔊
- Requests 🌐

---
**How to run the project in Vs code **
**1 Install dependencies**
pip install streamlit requests pillow pyttsx3
**2 Install Ollama (Important)**

Download Ollama:
👉 https://ollama.com

**Then run model:**

ollama run tinyllama

**🚀 Run Project**
streamlit run app.py
